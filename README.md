# Market Hub E-Commerce Web Application 🛒

## Overview

Market Hub is a full-stack E-Commerce web application developed using **Java, Spring Boot, HTML, CSS, and JavaScript**. The application provides users with a smooth online shopping experience, including product browsing, cart management, user authentication, and order handling.

## Features 🚀

* User Registration and Login Authentication
* Product Listing and Search Functionality
* Add to Cart and Remove from Cart
* Order Placement and Checkout System
* Admin Product Management
* Responsive User Interface

## Technologies Used 💻

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Java
* Spring Boot
* JDBC / JPA / Hibernate

### Database

* MySQL / PostgreSQL

## Architecture 🏗️

The project follows the **MVC (Model-View-Controller)** architecture for better code organization, scalability, and maintainability.

## Core Java Concepts Used ☕

* Object-Oriented Programming (OOP)
* Collections Framework
* Exception Handling
* JDBC Connectivity
* Multithreading Concepts

## Project Modules 📦

1. User Module
2. Product Module
3. Cart Module
4. Order Module
5. Admin Module

## How to Run the Project ▶️

1. Clone the repository
2. Open the project in Eclipse / IntelliJ IDEA
3. Configure the database in `application.properties`
4. Run the Spring Boot application
5. Open the browser and access the application
